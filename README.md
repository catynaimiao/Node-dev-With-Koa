# Node-dev With Koa

a quick setup backend framework for Nodejs Koa starter.

## Features

- typescript
- dotnev
- mongodb
- cors
- logger
- router with access control
- middleware etc.
- a basic example with Mongoose Schema (User).

## Scripts

```shell
npm i 
```

```shell
npm run start
```

then just edit the app.ts and request backend.
